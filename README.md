# 🗂️ 智能文件整理工具 (File Organizer)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一键自动整理你的混乱文件夹！支持按类型、日期、大小分类，还能自动重命名。

## ✨ 一键整理，告别混乱

整理前 vs 整理后：

```
Downloads/              organized_files/
├── photo.jpg            ├── 📁 images/
├── report.pdf           │   ├── photo1.jpg
├── video.mp4            │   └── screenshot.gif
├── old_doc.docx         ├── 📁 documents/
└── backup.zip           │   ├── report.pdf
                         │   └── report_20240128.pdf
                         ├── 📁 videos/
                         │   └── video.mp4
                         └── 📁 archives/
                             └── backup.zip
```

## 🚀 快速开始

```bash
# 按类型整理（推荐）
帮我整理这个文件夹，按文件类型分类

# 按日期整理
按修改日期整理文件，按月分类

# 整理 + 加序号
把大文件和小文件分开，加上序号
```

## 🎯 核心功能

| 功能 | 说明 | 适用场景 |
|------|------|----------|
| 🗂️ **智能分类** | 按类型/日期/大小/前缀自动分组 | 整理Downloads |
| ✏️ **自动重命名** | 时间戳、序号、保持原名 | 文件太多分不清 |
| 🔒 **安全模式** | 默认复制不删除，冲突自动处理 | 怕搞丢文件 |

## 📂 文件冲突处理

当目标文件已存在时，可选择：

- **跳过（默认）** - 保留目标文件，跳过当前文件
- **覆盖** - 用源文件替换目标文件
- **自动重命名** - 自动添加后缀 `_1`, `_2`, `_3`...
- **询问用户** - 每次冲突时询问如何处理

## 🖥️ 技术细节

- ✅ 支持 Windows、Linux、macOS
- ✅ 使用 bash 命令进行文件操作
- ✅ 支持嵌套目录结构
- ✅ 最大重命名尝试次数：1000

## 📋 配置选项

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 源文件夹路径 | 需要整理的目录 | - |
| 目标文件夹路径 | 整理后存放的目录 | organized_files |
| 分类方式 | 扩展名/大小/日期/前缀 | 扩展名 |
| 重命名方式 | 时间戳/序号/保持原名 | 保持原名 |
| 操作类型 | 复制/移动 | 复制 |
| 冲突处理 | 跳过/覆盖/重命名/询问 | 跳过 |

## ⚠️ 注意事项

1. **移动模式** - 使用移动模式会删除原文件，请谨慎操作
2. **覆盖模式** - 选择覆盖会导致同名文件被永久替换
3. **权限问题** - 确保对源目录和目标目录有读写权限
4. **特殊字符** - 文件名包含特殊字符时建议使用自动重命名
